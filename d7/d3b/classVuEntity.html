enum VuEntity::VU_SEND_TYPE

action to take for a created local unit inserted into DB

Enumerator:
VU_SC_DONT_SEND 

dont send creation, since each session reads its own from files (non volatiles)

VU_SC_SEND 

send creation, reliable but no hurry

VU_SC_SEND_OOB 

send creation, hurry (OOB)
