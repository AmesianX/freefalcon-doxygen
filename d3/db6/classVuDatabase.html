Detailed Description

VU main database interface. Every VuEntity in VU should be present here. Once inside, an entity is considered active (VU_MEM_ACTIVE). When removed, its marked as to be collected (VU_MEM_INACTIVE). After garbage collection or removal, its marked as colected (VU_MEM_REMOVED). Each active entity has a different ID, no active duplicates allowed. VuDatabase uses vuCollectionManager to update registered collections.
