Detailed Description

as name says, responsible for managing all registered collection. It calls Handle for all collections when a message is processed. It removes entities from all collections when database removes them. It can also insert entities in collections when entities are inserted into database. It can also clean all collections. Used by the VuDatabase class (insertion/removal) and also the VuMainThread::Update (birth list and garbage collector iteration).
